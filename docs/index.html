This single page application will enable nearly all waypoints for your Diablo 2 character in normal level and nightmare level.

It was created to test a character when using the -act5 command line switch.
For Diablo 2 Resurrected this option is not available.
You should enable the checkbox below instead to bring a characters stats up to Level 33

Ensure you have activated at least 1 way point before trying to apply this patcher.
Also your .d2s file must be at least 1024 bytes in size, so place some items in your stash or inventory first

Only the Halls of Pain waypoint is not enabled to allow repeated access to the portal leading to Nihlathak's Temple.
To use this tool, simply drag and drop your character file (ending in .d2s) into the white box below. Save the new file with the same name and replace the old file.

Ensure you keep a backup copy of your old character file.
