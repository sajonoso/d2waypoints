This single page application will enable nearly all waypoints for your Diablo 2 character in normal level and nightmare level.
To use this tool, Drag and drop your character .d2s file in the orange box below, your browser should then prompt you to save a new file.

Ensure you keep a backup copy of your old character file.
Use at your own risk. There is no warranty of any kind, either expressed or implied




This application was created to test a character when using the -act5 command line switch.
For Diablo 2 Resurrected this option is not available.
Instead, you should enable the checkbox to bring a characters stats up to Level 33

The make millionaire option will set character stash to 1,000,000 gold, inventory to 300,000 gold

This patcher can only modify existing attributes so please ensure you have some numbers in the character.
The .d2s file does not store anything with a zero value, so ensure something is not zero to allow the patcher to increase it.
For example:

  • * Leave some skills & attribute points unused to allow patcher to increase it.
  • * Ensure you have activated at least 1 way point before trying to apply this patcher
  • * To increase gold, make sure you already have some in the inventory and stash
Also your .d2s file must be at least 1024 bytes in size, so place some items in your stash or inventory first

Only the Halls of Pain waypoint is not enabled to allow repeated access to the portal leading to Nihlathak's Temple.
