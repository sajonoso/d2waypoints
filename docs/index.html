This single page application will enable nearly all waypoints for your Diablo 2 character in normal level.

It was created to test a character when using the -act5 command line switch

Only the Halls of Pain waypoint is not enabled to allow repeated access to the portal leading to Nihlathak's Temple.
To use this tool, simply drag and drop your character file (ending in .d2s) into the white box below. Save the new file with the same name and replace the old file.

Ensure you keep a backup copy of your old character file.
